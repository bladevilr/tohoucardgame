## 一、 核心战斗公式 (Core Formulas)

### 1.1 基础产分 (Scoring)
当菜品冷却结束“上菜”时：
```math
最终得分 = (基础味道 + 关键词修正 - 环境减益) × 技法乘区 × 评委偏好
```
*   **基础味道 (Flavor)**: 卡牌面板上的攻击力。
*   **技法乘区 (Technique)**: 全局属性。`系数 = 1.0 + (技法值 × 0.01)`。即 100 技法 = 2.0 倍率。
*   **环境减益**: 每一层【油腻】使味道 -2。

### 1.2 卖相压制 (Presentation Dominance - DoT)
这是“气场”造成的持续得分，每秒结算一次：
```math
每秒获得分数 = max(0, 我方卖相总值 - 对手卖相总值) × 0.5
```
*   **卖相 (Presentation)**: 卡牌面板上的防御力/血量总和。
*   **设计意图**: 鼓励堆叠卖相来建立“大势所趋”的优势，对抗高爆发的对手。

### 1.3 冷却加速 (Aroma Haste)
```math
冷却缩减 % = min(30%, 香气总值 × 0.5%)
```
*   **香气 (Aroma)**: 每 2 点香气 = 1% 冷却缩减。
*   **上限**: 最高 30% (需要 60 点香气)。

## 七、 经济系统 (Economy)

*   **初始金币**: 10G
*   **每日收入**: 10G + (回合数 x 1)
*   **利息**: 无利息机制 (鼓励消费)。
*   **刷新**: 2G
*   **升级**:
    *   3张同名卡 -> **2星** (属性 x 1.8，效果增强)
    *   3张2星卡 -> **3星** (属性 x 3.5，获得特殊词条)
